# Données

Le jeu de données d'entraînement (`fraudTest.csv`, ~150 Mo) n'est pas versionné :
il est trop volumineux pour le dépôt et reste sur la machine locale.

Placez le fichier dans ce dossier, ou ailleurs, et renseignez son chemin absolu
dans la variable `CSV_PATH` du fichier `.env`.

Source : jeu public *Credit Card Transactions Fraud Detection* (Kaggle / S3).
Seul `fraudTest.csv` est utilisé pour l'entraînement hors-ligne.
